Django Startup

=== How to Start a New Project ===

* STEP 1 - If you haven't installed 'cookiecutter', install it by

```
$ pip install cookiecutter
```

* STEP 2 - Start a new project by

```
$ cookiecutter https://github.com/panuta/django-startup.git
```
